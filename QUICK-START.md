## 🚀 立即部署到 Vercel

### 方法 1: 一键部署 (最快)

1. 访问 https://vercel.com/new
2. 粘贴这个仓库 URL (推送到 GitHub 后)
3. 点击 Deploy
4. 等待 2-3 分钟，完成！

### 方法 2: 使用 Vercel CLI

```bash
# 安装 CLI
npm i -g vercel

# 登录并部署
vercel
```

创建新文件 `content/2026-02-07.md`:

```markdown

date: 2026-02-07

  - Claude 3.6 发布

  - OpenAI 新产品
  - 热门开源项目

  - 新的教育工具

内容...
```
